Принцип работы:
на одном компьюетере (сервер) запускается server_gui.exe это принимающий компьютер, дальше отправляется "вредоносный код (virus_detect.exe)" на второй компьютер
1) запускаем server_gui.exe и ожидаем пока жертва запустит virus_detect.exe
2) на рабочем столе появится папка data с .json файлом в котором будет содержаться большое количество информации о системе жертвы

Важные особенности работы:
- на каждом из компьютеров должны быть установлены psutil и requests (pip install psutil requests)
- при первом использовании нужно убедиться что оба компьютера находятся в одной локальной сети
- предварительно указать локальный IP адресс принимающего компьютера в коде client.py и сформировать новый .exe с помощью команды pyinstaller --onefile --noconsole --name virus_detect client.py
